# XDD Core 词表

## 认证

- `session`
  登录后由服务端保存的一段会话信息。前端拿它判断当前人是不是已经登录。
- `fifa.owner`
  Fifa 的后台权限名。拿到这个权限的人可以进入 Fifa 的后台页面。
- `bobo.visitor`
  Bobo 的访客权限名。拿到这个权限的人可以进入 Bobo 相关页面。

## 路由

- `公开路由`
  不需要先登录就能打开的页面。
- `受保护路由`
  需要先确认当前登录状态和权限才能打开的页面。
- `无权限`
  已经登录，但没有对应权限，不能继续进入目标页面。

## 内容

- `内容`
  Fifa 后台里管理站点内容的菜单分组。第一版只放文章。
- `文章`
  Bobo 站点最终展示的一篇内容。正文使用 MDX 源码，可以先保存草稿，再发布给公开页面读取。
- `素材`
  文章里使用的图片文件。第一版只在文章编辑时上传和使用，不单独作为后台页面。
- `MDX 源码`
  文章正文的写法。它可以写普通 Markdown 语法，也可以插入已允许的组件。
- `草稿`
  文章还没有发布给公开页面读取的内容版本。
- `发布`
  把当前草稿变成 Bobo 公开页面可读取的文章内容。
- `预览`
  在文章发布前查看当前草稿在 Bobo 中的展示效果。预览不代表文章已经公开。
