# 后台底座优先 V1 验收清单

## 1. 模型边界验收

- [ ] `User` 只承载平台级资料，不承载博客业务字段
- [ ] `Role` 不再承担角色继承语义
- [ ] `Permission` 只用于最小权限集注册
- [ ] `UserRole` 为唯一用户授权入口
- [ ] `RolePermission` 不再暴露动态编辑平台语义

## 2. 权限验收

- [ ] 权限集合已收口为 V1 最小权限集
- [ ] 旧权限常量已删除
- [ ] `superAdmin` 具备稳定的全权限语义
- [ ] `admin` 权限仅覆盖后台管理需要
- [ ] `user` 权限仅覆盖自助资料能力

## 3. 接口验收

- [ ] 注册、登录、登出、获取会话接口可用
- [ ] 当前用户资料查询与更新接口可用
- [ ] 用户列表、用户详情、用户更新、用户状态接口可用
- [ ] 当前用户角色、当前用户权限接口可用
- [ ] 指定用户角色、指定用户权限接口可用
- [ ] 用户角色分配、移除接口可用
- [ ] 角色列表接口可用

## 4. 废弃接口清理验收

- [ ] 角色创建接口已移除
- [ ] 角色更新接口已移除
- [ ] 角色删除接口已移除
- [ ] 角色继承管理接口已移除
- [ ] 权限创建接口已移除
- [ ] 权限更新接口已移除
- [ ] 权限删除接口已移除
- [ ] 角色权限动态编辑接口已移除
- [ ] 用户创建接口已移除
- [ ] 用户删除接口已移除
- [ ] 用户角色缓存刷新接口已移除

## 5. 用户生命周期验收

- [ ] 默认管理语义不再使用硬删除
- [ ] 用户停用通过状态表达
- [ ] 用户封禁通过状态表达
- [ ] 若保留 `deletedAt`，其查询和文案语义已统一

## 6. 代码清理验收

- [ ] 与角色继承相关的旧代码已删除
- [ ] 与动态权限平台相关的旧代码已删除
- [ ] 与旧权限体系相关的旧 seed 已删除或重写
- [ ] 与旧设计冲突的注释和文案已删除或改写
- [ ] 不存在兼容旧语义的临时分支实现

## 7. 文档与测试验收

- [ ] README 与设计文档语义一致
- [ ] RBAC 文档已按 V1 收口更新
- [ ] OpenAPI 文案已更新
- [ ] 测试已覆盖当前角色与权限边界
- [ ] 验收项可通过实际接口行为验证

## 8. 最终验收结论

- [ ] 本次为“直接收口式重构”，无兼容层残留
- [ ] 当前代码库已符合“后台底座优先 V1”定位
