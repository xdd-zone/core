# 常用技能

这份文档只写当前仓库常用的技能顺序。

## 文档任务

先用：

- `xdd-plain-docs`

适用范围：

- README。
- `docs/`。
- 接口说明。
- JSDoc。
- 代码注释。
- 错误提示。
- 脚本说明。

## Nexus 任务

只要目标在 `apps/nexus`，先用：

- `xdd-honojs`

适用范围：

- Hono app。
- 接口。
- 路由。
- 模块。
- 插件。
- 中间件。
- 测试。

## Console 界面任务

先读：

- `apps/console/design-context.md`

再用：

- `frontend-design`

适用范围：

- 页面。
- 布局。
- 导航。
- 展示型组件。

## Console 界面加文案任务

顺序：

1. 读 `apps/console/design-context.md`
2. 用 `frontend-design`
3. 用 `xdd-plain-docs`

## Bobo 界面任务

先读：

- `docs/apps/bobo.md`

再用：

- `frontend-design`

适用范围：

- `apps/bobo/app` 页面。
- `apps/bobo/app` 布局。
- `apps/bobo/app/styles` 样式。
- `apps/bobo/components` 展示组件。

## Bobo 界面加文案任务

顺序：

1. 读 `docs/apps/bobo.md`
2. 用 `frontend-design`
3. 用 `xdd-plain-docs`
